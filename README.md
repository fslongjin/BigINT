# BigINT

这是一个使用C++语言实现的，基于单向链表的大整数类。

这个类支持将short、int、long long转换为BigInt，也支持将字符串转换为BigInt

重载了以下操作符：

- 输入输出操作符 >> <<
- 加减乘除运算符
- 比较运算符

并且还实现了乘方pow方法。
